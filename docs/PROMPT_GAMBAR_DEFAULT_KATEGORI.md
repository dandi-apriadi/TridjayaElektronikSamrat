# Prompt Gambar Default Per Kategori Produk

## Konteks

Toko Tridjaya Manado memiliki **68 kategori produk** dengan total ~3.200+ produk. Saat ini hampir semua produk menggunakan placeholder image generik (logo toko atau gambar kategori lain yang tidak sesuai). Hanya ~23 produk yang punya foto asli.

**Tujuan:** Membuat 1 gambar default per kategori yang representatif, modern, dan konsisten secara visual untuk ditampilkan di katalog online.

---

## Spesifikasi Teknis Gambar

- **Format:** WebP (untuk performa web)
- **Resolusi:** 800x800px (square, untuk konsistensi grid)
- **Background:** Gradient putih ke abu-abu muda (#f8f9fa → #e9ecef), clean & minimal
- **Style:** Product photography style, centered object, soft shadow, no text overlay
- **Lighting:** Studio lighting, soft diffused, slight reflection on surface
- **Naming convention:** `default-{slug-kategori}.webp` (contoh: `default-sepeda-listrik.webp`)

## Catatan Implementasi

Setelah gambar di-generate:

1. Simpan semua file di `frontend/public/assets/images/defaults/` dengan format `default-{slug}.webp`
2. Update `getImageUrl()` di `frontend/src/utils/apiClient.ts` untuk mapping kategori → gambar default
3. Atau update database langsung: 
   ```sql
   UPDATE products SET image = '/assets/images/defaults/default-{slug}.webp' 
   WHERE image IS NULL OR image = '' OR image LIKE '/assets/images/logo.webp';
   ```

## Tool yang Disarankan

- **Midjourney** — prompt di atas sudah dioptimalkan untuk Midjourney v6
- **DALL-E 3** — bisa langsung copy-paste prompt di atas
- **Stable Diffusion** — tambahkan negative prompt: `text, watermark, logo, brand name, blurry, low quality, distorted`
- **Adobe Firefly** — cocok untuk product photography style

Untuk konsistensi, generate semua gambar menggunakan tool yang sama.
