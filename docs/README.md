## Prinsip Utama

- Frontend publik harus cepat, ringan, dan SEO-friendly.
- Backend harus efisien, terukur, dan mudah diaudit.
- Semua data sensitif harus dilindungi dengan kontrol akses ketat.
- Telemetri dan tracking harus patuh pada prinsip minimasi data.
- Setiap komponen harus gagal dengan aman dan tidak membuka celah eksploitasi.
